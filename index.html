üìê Performance Metrics Explained

Understanding the evaluation criteria

Annual Return

The annualized rate of return over the investment period. Calculated as: ((Final Value / Initial Value)^(1/years) - 1) √ó 100. Higher values indicate better performance. PPO achieved the highest at 27.14%.

Sharpe Ratio

Measures risk-adjusted return by comparing the portfolio's excess return to its volatility. Formula: (Return - Risk-Free Rate) / Standard Deviation. Higher values indicate better risk-adjusted performance. DDPG showed the best Sharpe ratio.

Sortino Ratio

Similar to Sharpe ratio but only considers downside volatility (negative returns), making it more relevant for investors concerned about losses. DDPG achieved the highest at 2566.93, indicating excellent downside protection.

Maximum Drawdown

The largest peak-to-trough decline in portfolio value during the investment period. Represents the worst-case loss an investor would have experienced. DDPG and MVO showed the best drawdown control at around ‚Çπ6-7L.
